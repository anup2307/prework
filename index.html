HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • CSS (Cascading Style Sheets) is used to style and lay out web pages.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The external CSS stylesheet is connected to the HTML file using the link element in the header section of the HTML code.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git pull origin main- it fetches from main branch and merges into the new branch
  • Git restore - discards changes in the working directory
  • Git add-A - adds all the modified files to the staging directory
  • Git commit -m "comments" - commits all the files in the staging directory
  • Git push origin "sub branch name" - creates pull request in the github repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • console.log is used to output a message to the web console by adding an argument
  • Script command is used to link the javascript file to the HTML file
